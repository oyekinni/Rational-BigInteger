Programming Exercise 20.11 – Match Grouping Symbols

This program checks whether a Java source file has correctly matched grouping symbols. 
It verifies parentheses (), braces {}, and brackets [].

The program uses a stack to track opening symbols and compares them with closing symbols.
If all symbols match correctly, it prints that they are matched. Otherwise, it shows an error.

How to run:

1. Compile:
   javac MatchGroupingSymbols.java

2. Run:
   java MatchGroupingSymbols TestFile.java
